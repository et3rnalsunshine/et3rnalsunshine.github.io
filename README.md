Rework my personal literary website into an elegant editorial-style homepage inspired by a refined literary magazine.

The website is for Sophia El Kerdini, a Moroccan writer, essayist, and book reviewer based in Germany. The site should feel calm, intelligent, literary, and timeless — not commercial, not influencer-like, not colorful or trendy.

Design style:
- Editorial magazine layout
- Warm ivory / off-white background
- Black or dark charcoal text
- Thin grey dividers
- Large elegant serif typography for titles
- Clean sans-serif for small navigation and metadata
- Lots of white space
- Moody literary photography: notebooks, books, writing desk, window light, sea, paper, dried flowers
- Minimal, structured, refined

Fonts:
- Use an elegant serif for headings, similar to Cormorant Garamond, EB Garamond, Playfair Display, or a Baskerville-like font.
- Use a clean sans-serif for navigation and small labels, such as Inter, Lato, or Libre Franklin.
